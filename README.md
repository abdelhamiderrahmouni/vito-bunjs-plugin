# Bun.js Plugin for VitoDeploy

Integrate Bun.js into your VitoDeploy instance with this plugin.

## Documentation

> [!WARNING]
> For now you can only install one version of Bun.js per VitoDeploy instance.
> And this plugin is only compatible with VitoDeploy v3.

### Features
- Install Bun.js globally on your VitoDeploy instance.
- Specify the version of Bun.js to install.
- Create new projects using Bun.js as the runtime.
- Create workflows that utilize Bun.js.

### Installation

To install the plugin, navigate to the `Admin -> Plugins` and select `Bun.js Plugin` in the Community tab.

### Uninstall
To uninstall the plugin, navigate to the `Admin -> Plugins` and you can find the `Bun.js plugin` in the Installed tab and you can uninstall it.
